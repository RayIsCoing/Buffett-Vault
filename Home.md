## How to Use | 使用指南

1. **阅读一封信** → 打开 `Sources/Shareholder Letters/` 对应年份的笔记 → 点击 PDF 链接阅读原文 → 在笔记中记录要点
2. **发现一个概念** → 用 `[[概念名]]` 创建链接 → 到 `Topics/` 里展开这个概念
3. **提到一个人/公司** → 同样用 `[[]]` 链接到 `People/` 或 `Companies/`
4. **看关系网络** → 打开图谱视图（左侧栏图标），看你的知识是如何连接的

> [!tip] 小贴士
> 不需要一次性读完所有信。每读一封，认真做笔记，用链接把知识串起来，比囫囵吞枣读完49封更有价值。
