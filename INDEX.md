## Search Tips for Agents

1. **Company analysis?** Start with Notes/Holdings-History.md for the buy/sell timeline, then grep shareholder-letters and annual-meetings for the company name.
2. **Philosophy question?** Start with Notes/Buffett-Philosophy-Evolution.md and Topics/*.md. Only grep primary sources if you need exact quotes.
3. **"What did Buffett say about X?"** Grep annual-meetings first (Q&A format = more candid), then shareholder-letters.
4. **Mistakes and regrets?** Start with Notes/Rejected-and-Regretted-Investments.md, then search for "mistake" or "error" in shareholder-letters.
5. **Quantitative data?** Use Attachments/sec-filings/*.csv for actual position sizes and timing.
6. **Budget:** 3-5 greps + 2-3 reads per query. Find what you need and stop.
